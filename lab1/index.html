  1. Open index.html in web-browser.
  2. Try to search for a baby animal. We think kittens are cute.
  3. Open the debug console in web-browser cmd+alt+i / ctrl+shift+i . Something is wrong...
    4. The request is not working since the page isn't served through a server. Why does this matter?
  4. Install Node if you haven't already. It's even better to install through Homebrew if you're on OSX. Windows users have to use the webpage.
  5. Time to get up and running with a development server!
    1. First order of business is to run npm init in root directory. Press enter through all the steps to accept the default values.
    2. Familiarize yourself with the package.json file and find the scripts section.
    3. Run npm install --save-dev http-server to add a simple development server as a development dependency. What is the difference between development dependencies and regular ones?
    4. To start our server we may want to add a new npm script. Create a new key/value pair under the script "test", "start": "http-server"
    5. Start the development server by running our shiny new script: npm run start in the terminal
    6. Go to http://127.0.0.1:8080
    7. Verify that the search is working. I can haz kittens?
    8. Familiarize yourself with index.html. and index.js
    9. Can you find in the markup where kitten images loaded? And how?
