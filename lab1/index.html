📖   Instructions

  1. Search for a baby animal, e.g. "kitten"
  2. Install Node (even better to install through Homebrew if on OSX)
  3. Up and running with a development server
    1. npm init in root directory. Press enter through all the steps to accept defaults
    2. Familiarize with package.json and find the scripts section.
    3. npm install --save-dev http-server (Adds a simple development server as a development dependency)
    4. Create a new key/value pair under "test", "start": "http-server"
    5. Start the development server by typing npm run start in the terminal
    6. Go to http://127.0.0.1:8080
